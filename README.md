# pypackmol

A python wrapper for packmol.

## Requirements

- [gfortran](https://gcc.gnu.org/wiki/GFortran)

    ```bash
    apt install gfortran
    ```

- [packmol](http://m3g.iqm.unicamp.br/packmol/home.shtml)

    ```bash
    ./build-packmol.sh
    ```

## Installation

```bash
pip install pypackmol
```
